# Zombie Dash

A 2D zombie survival game built with **Unity** and **C#**.  
This repository contains the full Unity project (Assets/Packages/ProjectSettings) along with project documentation from a semester-long software engineering course.

---

## Overview

Zombie Dash is a fast-paced survival game where the player must navigate a zombie-infested environment and stay alive as long as possible.

The objective is to:

* Avoid contact with zombies
* Survive through increasing difficulty
* Use movement and positioning to stay out of danger
* Stay alive as long as possible

The game was developed iteratively using sprint cycles with regular demos and feedback-driven improvements.

---

## Features

* **Core Gameplay Mechanics**
  + 2D player movement and controls
  + Zombie enemy behavior (AI)
  + Collision detection and damage/health handling
  + Survival-style gameplay loop and progression
* **Unity-Based Architecture**
  + Component-based object design
  + Scene management and game state control
  + Script-driven gameplay logic written in C#
* **Agile Development Process**
  + User stories and scenario documentation
  + Sprint-based development across ~5 months
  + Live demos with 50+ testers
  + Iteration based on feedback
* **Team Collaboration**
  + Git version control with feature branches
  + PR-style merging and merge conflict resolution
  + Organized sprint artifacts and documentation included in the repo
